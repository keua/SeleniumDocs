# NightwatchJS

Is a NodeJS based framework that uses the Selenium webdriver to do the tests.

## Requirements

> * Java JDK 8, ref. [install java](https://www.digitalocean.com/community/tutorials/how-to-install-java-on-centos-and-fedora)
> * NodeJS, 
> * npm
> * webdriver-manager
> * nightwatch
> * Mozilla Firefox
